Настройки config.json
zapros => Какие запросы будет парсить скрипт, если хотите один ["zapros"], если больше, то добавляйте запросы по аналогии как и второй через запятую и пробел
parse_zapros => Принимает булево значение false не парсить запросы true парсить (по умолчанию false)
email_parse => Парсить ли email:pass строчки (по умолчанию true)
login_parse => Парсить ли login:pass строчки (по умолчанию true)
number_parse => Парсить ли number:pass строчки (по умолчанию true)
parse_full => Парсить ли полную строку false нет true да (по умолчанию false)
bad_char => Символы которые не попадут при проверке в login и number
folder => Папка в которую будут сохранён результат (по умолчанию Result)
size => Какой предел элементов в словаре будет перед его очисткой? (по умолчанию 10000)
file_block => Сколько будет мб в блоке при чтении файла (по умолчанию 256мб)
